## About Sunburst Dashboard

Sunburst is a modern web application built with Laravel and Vite, featuring real-time data handling, custom UI components powered by Alpine.js, and smooth animations using GSAP.

---

## Environment & Compatibility Versions

To ensure smooth installation and prevent version conflict errors (`ERESOLVE`), please ensure your environment matches or uses compatible versions:

* **PHP:** ^8.2 (Recommended: 8.5+)
* **Composer:** Latest stable version
* **Node.js & npm:** Node.js v18+ / v20+ recommended
* **Laravel Framework:** v13.20.0
* **Vite Plugin Laravel:** v1.3.0
* **Vite:** v8.1.4

---
